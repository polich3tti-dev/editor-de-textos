
# Editor de Textos 


## Funcionalidades:
- texto em negrito
- texto em itálico
- alinhar texto na esquerda
- alinhar texto no centro
- alinhar texto na direita
- transformar texto em maiúsculo
- transformar texto em minúsculo
- transformar primeira letra em maiúsculo
